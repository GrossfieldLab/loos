39 string fullHelpMessage(void) {
40 
41  string full_help_message =
42 "SYNOPSIS\n"
43 " Assign backbone suites to RNAs based on backbone dihedrals.\n"
44 "\n"
45 "DESCRIPTION\n"
46 " The goal of this tool is to assign continuous RNA dinucleotides to a\n"
47 " cluster called a \"suite\" based on the conformation of backbone dihedrals.\n"
48 " The idea comes from Richardson et al. (2008) RNA 14, 465-481. The\n"
49 " dinucleotide for a residue runs from delta (C5'-C4'-C3'-O3') of the previous\n"
50 " residue to delta of the current residue, encompassing seven continuous\n"
51 " dihedrals. A suite is a pre-defined cluster in this 7D space, named by a\n"
52 " two-character string. Examples are \"1a\" or \"5z\".\n"
53 "\n"
54 " The first step is to search the given selection for RNA backbone atoms, i.e.\n"
55 " atoms named \"P\", \"O5'\", \"C5'\", \"C4'\", \"C3'\", or \"O3'\". These atoms are\n"
56 " split by residue. Valid dinucleotides are sets of delta-to-delta backbone\n"
57 " atoms with sequential resids. Once the set of valid dinucleotides is\n"
58 " determined, the tool will loop over the trajectory and assign each\n"
59 " dinucleotide to a suite for each frame.\n"
60 "\n"
61 " Suite assignment occurs in two stages. The clusters are well-separated in\n"
62 " the 3D subspace of delta(i-1), delta, and gamma. So the first stage is to\n"
63 " assign each delta to one of two ranges of values consistent with either a\n"
64 " C3'-endo (3) or C2'-endo (2) sugar pucker and to assign gamma to one of\n"
65 " three ranges of values: gauche plus (p), gauche minus (m), or trans (t). The\n"
66 " result is a three-character string called a ddg index. Examples are \"33p\"\n"
67 " or \"23t\". Then, the dinucleotide is assigned to one of a possible set of\n"
68 " suites associated with its ddg index based on a scaled hyperellipsoid\n"
69 " distance in the dual 4D subspace of epsilon, zeta, alpha, and beta.\n"
70 "\n"
71 " Some suites have overlapping hyperellipsoids of different sizes. The wider\n"
72 " suite is called a dominant suite, and the narrower suite is called a\n"
73 " satellite suite. These cases are handled by rescaling the hyperellipsoid\n"
74 " distance along the dimensions in which the overlap occurs.\n"
75 "\n"
76 " If a dinucleotide doesn't fit into one of the allowed ranges for a dihedral,\n"
77 " it is assigned as an outlier and given a suite name \"!s\", where \"s\" is the\n"
78 " first character of the name of the deviant dihedral, e.g. \"!a\" for a bad\n"
79 " alpha. If the dinucleotide is not close to any of the reference suites, it \n"
80 " is also assigned as an outlier and given a suite name \"!!\".\n"
81 "\n"
82 " After assignment, each dinucleotide is given a goodness-of-fit score called \n"
83 " the suiteness based on the scaled 7D hyperellipsoid distance to its assigned\n"
84 " suite. A suiteness of one indicates that the dinucleotide is at the cluster\n"
85 " center. Lower suiteness indicates that the dinucleotide is farther from the\n"
86 " cluster center. An outlier has a suiteness of zero, and assigned\n"
87 " dinucleotides have a minimum suiteness score (set by the -c option) to\n"
88 " differentiate them from outliers.\n"
89 "\n"
90 " It is necessary to specify a path to a file containing definitions for the\n"
91 " reference suites on the command-line. The format is explained in the next\n"
92 " section. An example of the format that implements the suites as defined in\n"
93 " the software suitename (Richardson et al. (2008) RNA 14, 465-481) is\n"
94 " included as share/suitename_definitions.dat in the top-level directory of\n"
95 " the LOOS source tree. If installing within a conda environment, this file\n"
96 " can also be found in $CONDA_PREFIX/share/loos/suitename_definitions.dat;\n"
97 " otherwise, it can be found in $LOOS/share/suitename_definitions.dat. The\n"
98 " suitename_defintions.dat file should be sufficient for typical users, but\n"
99 " you must specify the path to it as the first positional argument.\n"
100 "\n"
101 "SUITE DEFINITON FILE FORMAT\n"
102 " Each line in the file is parsed as a record containing fields with a width\n"
103 " of eight characters. Blank lines and lines beginning with \"#\" are ignored.\n"
104 " The first field specifies the type of record and must be one of \"suite\",\n"
105 " \"width\", \"domsat\", \"delta\", \"epsilon\", \"zeta\", \"alpha\", \"beta\", or \"gamma\".\n"
106 " These records and their associated fields are described below.\n"
107 "\n"
108 " suite name ddg delta(i-1) epsilon zeta alpha beta gamma delta(i)\n"
109 " Define a reference suite with suite name given in field 2, ddg index\n"
110 " given in field 3, and dihedrals of the cluster center given in fields 4\n"
111 " through 10.\n"
112 "\n"
113 " width delta(i-1) epsilon zeta alpha beta gamma delta\n"
114 " Define default widths for scaled hyperellipsoid distances.\n"
115 "\n"
116 " domsat sat_name dom_name dihedral_index sat_width dom_width\n"
117 " Define dominant-satellite pair with name of satellite suite in field 2,\n"
118 " name of dominant suite in field 3, index of dihedral dimension with\n"
119 " altered width in field 4, width of that dimension for satellite suite\n"
120 " in field 5, and width of that dimension for dominant suite in field 6.\n"
121 " Additional dimensions and widths can be specified in fields 7 through 9,\n"
122 " fields 10 through 12, etc.\n"
123 "\n"
124 " dihedral min max\n"
125 " Define allowed ranges for a dihedral. \"dihedral\" can be one of \"delta\",\n"
126 " \"epsilon\", \"zeta\", \"alpha\", \"beta\", or \"gamma\". The minimum value\n"
127 " is given in field 2 and maximum value in field 3.\n"
128 "\n"
129 "EXAMPLES\n"
130 " rna_suites $CONDA_PREFIX/share/loos/suitename_defintions.dat foo.pdb foo.dcd\n"
131 " Assign backbone suites using the install prefix from a conda install.\n"
132 "\n"
133 " rna_suites -s 'resid <= 10' $CONDA_PREFIX/share/loos/suitename_defintions.dat \\\n"
134 " foo.pdb foo.dcd\n"
135 " Assign backbone suites only for the first 10 residues.\n"
136 "\n"
137 " rna_suites -c 0.001 $CONDA_PREFIX/share/loos/suitename_defintions.dat \\\n"
138 " foo.pdb foo.dcd\n"
139 " Assign backbone suites using a minimum suiteness of 0.001 for\n"
140 " non-outliers.\n";
141 
142  return full_help_message;
143 
144 }

178 int main(int argc, char *argv[]) {
179 
180  // Get command-line input
181  string header = invocationHeader(argc, argv);
182 
183  // Set up tool options
184  opts::BasicOptions *bopts = new opts::BasicOptions(fullHelpMessage());
185  opts::BasicSelection *sopts = new opts::BasicSelection("!hydrogen");
186  opts::RequiredArguments *ropts = new opts::RequiredArguments;
187  ropts->addArgument("suite_def", "suite_definition_file");
188  opts::TrajectoryWithFrameIndices *tropts =
189  new opts::TrajectoryWithFrameIndices;
190  ToolOptions *topts = new ToolOptions;
191 
192  opts::AggregateOptions options;
193  options.add(bopts).add(sopts).add(ropts).add(tropts).add(topts);
194  if (!options.parse(argc, argv))
195  exit(-1);
196 
197  // Assign tool options to variables
198  const double suiteness_cutoff = topts->suiteness_cutoff;
199 
200  // Print command-line input
201  cout << "# " << header << endl;
202 
203  // Build LOOS system and generate atom selection
204  AtomicGroup model = tropts->model;
205  pTraj traj = tropts->trajectory;
206  vector<uint> indices = tropts->frameList();
207  AtomicGroup rna_atoms = selectAtoms(model, sopts->selection);
208 
209  // Create RNASuite object from RNA atoms
210  string suite_definition = ropts->value("suite_def");
211  RnaSuite rna_suite = RnaSuite(rna_atoms, suite_definition, suiteness_cutoff);
212  vector<int> suite_resids = rna_suite.getSuiteResids();
213  vector<string> suite_resnames = rna_suite.getSuiteResnames();
214  //rna_suite.printReferenceSuites();
215 
216  // Print dihedrals
217  //rna_suite.printBackboneAtoms();
218 
219  // Print column headers
220  cout << "# Frame Resid Resname Suite DDG_index Suiteness" << endl;
221 
222  // Loop over trajectory
223  vector<string> suite_names;
224  vector<string> suite_ddgs;
225  vector<double> suiteness;
226  uint t = 0;
227  for (vector<uint>::iterator i = indices.begin(); i != indices.end(); ++i) {
228 
229  traj->readFrame(*i);
230  traj->updateGroupCoords(model);
231 
232  rna_suite.calculateBackboneDihedrals();
233  rna_suite.assignSuitenameSuites();
234  suite_names = rna_suite.getSuiteNames();
235  suite_ddgs = rna_suite.getSuiteDDGs();
236  suiteness = rna_suite.getSuitenessScores();
237 
238  for (uint j = 0; j < suite_resids.size(); ++j)
239  cout << boost::format("%5d %5d %3s %2s %2s %8.6f") % t
240  % suite_resids[j] % suite_resnames[j] % suite_names[j]
241  % suite_ddgs[j] % suiteness[j] << endl;
242 
243  ++t;
244 
245  }
246 
247 }
